스트레스 요실금에서 Mesh 치료 원리

```

1️⃣ 정상인 - 평상시

✅ 골반저 근육이 튼튼
✅ 요도가 잘 지지됨
✅ 정상 배뇨 가능

1️⃣ 정상인 - 기침할 때

✅ 복압 ↑ → 방광 눌림
✅ 골반저가 요도를 받쳐줌
✅ 요도가 닫혀서 소변 안 샘
정상인의 핵심: 기침할 때 복압이 증가해도 튼튼한 골반저 근육이 요도를 받쳐줘서 요도가 닫힙니다!

2️⃣ 스트레스 요실금 환자 - 평상시

⚠️ 골반저 근육이 약함
⚠️ 요도가 아래로 처짐
✅ 평상시는 배뇨 가능

2️⃣ 스트레스 요실금 환자 - 기침할 때

❌ 복압 ↑ → 방광만 눌림
❌ 골반저가 약해서 지지 안 됨
❌ 요도가 안 닫혀서 소변 샘!
문제의 핵심: 골반저가 약해서 기침할 때 요도를 받쳐주지 못합니다. 그래서 요도가 안 닫히고 소변이 샙니다!

3️⃣ Mesh 수술 후 - 평상시

✅ Mesh가 느슨하게 위치
✅ 요도를 압박 안 함
✅ 정상 배뇨 가능

3️⃣ Mesh 수술 후 - 기침할 때

✅ 복압 ↑ → 요도가 아래로 눌림
✅ Mesh가 받쳐줘서 요도가 닫힘
✅ 소변 안 샘!
🎯 Mesh의 핵심 원리:
  • 평상시: Mesh가 느슨하게 있어서 → 요도 안 막음 → 정상 배뇨 가능
  • 기침할 때: 복압으로 요도가 Mesh에 눌림 → 요도 닫힘 → 소변 안 샘
  • 결론: Mesh는 약해진 골반저 근육을 대신해서 "받침대" 역할을 합니다!

📌 정리

Mesh는 괄약근을 조이는 게 아닙니다!
대신 "해먹(hammock)"처럼 요도 아래에 받침대를 만들어서,
기침할 때 복압이 증가하면 요도가 Mesh에 눌려서 일시적으로 닫히게 만듭니다.

평상시엔 느슨해서 배뇨에 지장 없고,
필요할 때만(복압 증가 시) 작동하는 "똑똑한 받침대"입니다! 🎯

```
